Introducing the Multiple Projects Exporter – a set of powerful tools that extend the default Unity build options.
Using this extension you can define different profiles and quickly export your project using these settings. This could be used to create anything from an alternate set of scenes or demo, to different standalone products managed in the same Unity project.
